## 介绍

智慧公交监控调度项目，建设整体监控调度系统，前端车载系统包含公交调度屏、监控主机、摄像机等，后端平台包括智能公交调度系统及视频综合管理系统，实现前端车辆智能排班、实时调度、视频监控等业务应用。

智慧公交提升运营效率。通过部署智能公交调度与客流统计系统，可根据线路客流量合理排班、调度公交车辆，提高企业运营效率，提升民众出行体验；

智慧公交保障安全。车载视频监控系统的部署，实现对于公交车内情况实时监控记录，车内违法犯罪行为得到有效威慑、控制和减少。
