## 1 功能简介

车头卡口基础特征识别摄像机应用于地面道路、城市快速路、高速公路等各种道路，抓拍车头方向，在NNIE模块强大算力的支持下，采用各类深度学习算法，实现车辆的检测、号牌检测与检测、车身颜色检测、车辆类型与品牌的检测等功能.。

| 摄像机类型 | 支持型号 | 所属版本 | 最大分辨率 | 最大监测车道数 | 主要功能                                                     |
| ---------- | -------- | -------- | ---------- | -------------- | ------------------------------------------------------------ |
| 摄像机     |          |          | 900万像素  | 3车道          | 1车辆检测抓拍2车牌号码检测3车牌颜色检测4车身颜色检测5车辆品牌检测 |
